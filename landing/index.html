How Engram manages
your agent's memory

Three bio-inspired layers working together to store what matters,
forget what doesn't, and organize knowledge dynamically.

FadeMem

Human-like forgetting with Ebbinghaus decay curves. Dual-layer SML/LML with automatic promotion. ~45% storage reduction.

Learn more

EchoMem

Multi-modal encoding with 5 retrieval paths: paraphrase, keywords, implications, question-form, and temporal markers.

Learn more

CategoryMem

Dynamic hierarchical categories that evolve with your data. Auto-discovered, self-organizing, with category decay.

Learn more

Get Started in 3 Lines

Terminal 
# Clone and install
git clone https://github.com/Engram-AI/Engram.git
pip install -e "./Engram[all]"

# Configure MCP for Claude Code / Cursor
engram-install

# Or use as Python library
from engram import Memory

memory = Memory(user_id="agent-1")
memory.add("User prefers Python over JavaScript")
results = memory.search("What programming language?")
